1. Row with gutters

Row gutters are specified globally as a % of the whole browser width. Row gutters can have different % values at each layout break.

1a. Container with outer gutters and inner gutters

Container gutters are specified globally as a % of the whole browser width. Containers can have different % values for outer and inner gutters. Container gutters can have different % values at each layout break.
